🎯 Project Overview
DEFEND.X is a unified AI-powered cybersecurity platform that fights AI-driven threats with AI. It specifically tackles Shadow AI data leaks (when employees use unchecked AI tools like ChatGPT, Claude, Gemini, etc.) while providing proactive attack simulation, real-time threat detection, and prompt-level protection — all in one dashboard.
Instead of traditional rule-based security, DEFEND.X uses an AI vs AI architecture:

Red AI simulates real attacks to train and test defenses.
Blue AI provides real-time anomaly detection.
Shadow AI Prompt Firewall blocks sensitive data leaks at the source (browser level).

No more fragmented tools. One platform for attack simulation → detection → prevention → response.

🚨 Problem Statement

Shadow AI data leaks caused by employees unknowingly pasting API keys, PII, source code, or proprietary data into public AI tools.
Lack of real-time threat detection leaving organizations blind to AI-assisted attacks.
Security teams are forced to juggle multiple fragmented tools — no single platform combines attack simulation + defense + prevention.
